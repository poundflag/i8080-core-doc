Decodes and executes the instruction based on the opcode.

This function takes the opcode and performs the corresponding instruction execution. It wont run the entire instruction but just the one machine cycle. It also updates the temporary address and halt signal if necessary.

Parameters
[in]opcodeThe opcode of the instruction.
[in]machine_cycleThe current machine cycle.
[out]temporary_addressThe temporary address that might be updated during execution.
[out]halt_signalThe halt signal that might be updated during execution.
Returns
True if the operation was successful, otherwise false.
