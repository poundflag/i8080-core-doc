Loads memory with an array of data.

This function loads an array of data into memory starting from the specified address offset.

Parameters
[in]file_dataPointer to the array of data.
[in]file_sizeThe size of the data array.
[in]address_offsetThe offset indicating where the data writing should begin. For example, an offset of 10 starts writing at address 10.
